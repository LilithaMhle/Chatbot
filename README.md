# Pink Chatbot 

**My personal chatbot friend with voice, text, and smart conversation.**  

---

## Description
Pink Chatbot is an intelligent, interactive chatbot built with Python (Flask) and styled with a peachy pink Glossmorphic theme. It can:  

- Respond via **text** or **voice**.  
- Detect **user emotions** using sentiment analysis.  
- Provide **sympathetic and friendly responses**, advice, and jokes.  
- Maintain a **conversation history** displayed as topics in the sidebar.  
- Be your **personal AI friend**, ready to chat anytime!  

---

## Features
- Cute chat bubbles with **pink** for chatbot messages and **burgundy** for your messages.  
- Glossmorphic design giving **shiny, modern vibes**.  
- Voice input using microphone.  
- Sidebar showing previous conversation topics automatically.  
- Responsive layout, works well on desktop screens.
